Система управления кофейней
Создайте консольное приложение для управления процессами в небольшой кофейне.
Приложение должно позволять:
• добавлять и удалять кофемашины;
• запускать приготовление напитков (например, эспрессо, капучино) на
выбранной кофемашине;
• пополнять запасы воды, молока и кофе в кофемашине;
• получать уведомление, если какая-либо из кофемашин не может выполнить
заказ из-за нехватки ресурсов.
Технические требования:
• Каждая кофемашина должна расходовать ингредиенты (воду, молоко, кофе) при
приготовлении напитков.
• Если в кофемашине недостаточно ингредиентов, она не должна начинать
приготовление и должна сообщать об этом.
• Взаимодействие с пользователем должно происходить через консольное
меню.
• Программа должна быть спроектирована с использованием нескольких
классов, отражающих разные сущности (например: Кофемашина,
МенеджерКофейни, Напиток и др.).

• При проектировании необходимо применить принципы объектно-
ориентированного программирования, включая агрегацию и/или композицию
между объектами.
